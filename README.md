## What is?

일반적으로 공인 인증기관은 사설 네트워크에 대해 IP인증서를 발급하지 않는다.
내부 시스템간 TLS 통신을 하기 위해서는 사설 인증서가 필요한데, 인증서를 편리하게 발급하기 위한 wrapper가 필요하여 개발하게 되었다.

## Features

- 사설 루트인증서 발행
- 사설 루트인증서를 기반으로 서버 인증서 발행
- 발행한 인증서 메타정보 관리
- 이용자 권한 차등화
